Sistema referidos

El presente proyecto consiste en desarrollar e implementar un aplicativo web para el manejo de datos de referidos en las campañas políticas, la principal característica de esta aplicación es la facilidad en el análisis de la información gracias a un sistema de centralizado y de capturas de datos en tiempo real, resolviendo así necesidades específicas de las campañas políticas. Se incluyen servicios como el registro de geolocalización en tiempo real, e informes detallados de toda la información obtenida

DOM

El DOM permite un acceso a la estructura de una página HTML mediante el mapeo de los elementos de esta página en un árbol de nodos. Cada elemento se convierte en un nodo y cada porción de texto en un nodo de texto, Document Object Model es el conjunto de objetos que un programa de Javascript puede actuar sobre ellos.

3 ejemplos practicos han sido realizar un programa para retirar dinero de un cajero, El dibujo del canvas simulando un lapiz y Mover con las teclas una imagen sobre un canvas.
